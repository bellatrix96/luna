Setup guide · ~10 minutes total
Step 1 of 3

Host Luna on GitHub Pages

Create a GitHub repository

  1. Go to github.com and sign in (or create a free account)
  2. Click the + icon → New repository
  3. Name it luna — keep it public (required for free GitHub Pages)
  4. Click Create repository

Upload your files

  1. In your new repo, click Add file → Upload files
  2. Upload the entire luna folder contents (drag all files and folders)
  3. Make sure index.html is in the root of the repo
  4. Click Commit changes

Enable GitHub Pages

  1. Go to Settings → Pages in your repository
  2. Under Source, select Deploy from a branch
  3. Choose main branch, folder / (root)
  4. Click Save
  5. Your Luna will be live at https://yourusername.github.io/luna in ~2 minutes
Step 2 of 3 · Optional but recommended

Google Calendar API Key

  1. Go to console.cloud.google.com
  2. Create a new project — name it Luna
  3. Go to APIs & Services → Library
  4. Search for Google Calendar API and click Enable
  5. Go to APIs & Services → Credentials
  6. Click Create Credentials → OAuth Client ID
  7. Application type: Web application
  8. Under Authorised JavaScript origins, add your GitHub Pages URL:
    https://yourusername.github.io
  9. Under Authorised redirect URIs, add:
    https://yourusername.github.io/luna/
  10. Click Create — copy your Client ID
  11. Paste it into Luna's Settings → API Keys → Google OAuth Client ID
⚠️ You'll also need to add yourself as a test user: APIs & Services → OAuth consent screen → Test users → Add your Gmail address
Step 3 of 3 · Optional

YouTube API Key (for song search)

  1. In the same Google Cloud project, go to APIs & Services → Library
  2. Search for YouTube Data API v3 and click Enable
  3. Go to Credentials → Create Credentials → API Key
  4. Copy the API key
  5. Click Restrict Key → under API restrictions, select YouTube Data API v3
  6. Also restrict by HTTP referrer: add https://yourusername.github.io/*
  7. Paste the key into Luna's Settings → API Keys → YouTube API Key
Free tier gives you 100 song searches per day — more than enough for personal use. The search is debounced so you don't burn quota while typing.
